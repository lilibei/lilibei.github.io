如何搭建WordPress博客平台

2019-12-06 | 博客搭建

  2. 2. 购买虚拟服务器（VPS）

  4. 4. 安装WordPress

购买域名

  • 登陆GoDaddy主页并注册账号
  • 输入想要注册的域名，并点击搜索域名（选择.online作为网站后缀因为便宜）

  • 加入并进入购物车

  • 选择支付方式

  • 查看购买结果

购买虚拟服务器（VPS）

  • 进入老薛主机主页购买美国一号套餐（便宜、稳定、快速）。

  • 输入上一步在GoDaddy主页购买的域名

  • 在年付优惠处打勾并点击继续

  • 输入优惠码验证后付款（优惠码百度即可,享受八折优惠）

  • 填写邮箱、QQ、电话等信息付款成功后可收到电子账单

  • 使用上一步的邮箱号登陆老薛主机的控制端

  • 登陆购买的服务器美国一号

  • 记录服务器IP地址（解析DNS会使用）

使用DNSPod解析DNS

  • 登陆GoDaddy账户后点击管理域名

  • 进入我的域名管理界面点击管理DNS

  • 找到域名服务器处点击更改

  • 点击输入我自己的域名服务器（高级）

  • 输入DNS地址f1g1ns1.dnspod.net和f1g1ns2.dnspod.net并点击保存

  • 登陆DNSPod主页点击右上角管理控制台

  • 点击左面功能栏我的域名并添加从GoDaddy购买的域名

  • 点击我的域名并按照红框的内容添加，蓝框内添加服务器的IP地址和购买到的域名

安装WordPress

  • 登陆购买的服务器美国一号并登陆cPanel面板

  • 找到语法功能分类栏并点击安装WordPress

  • 填写安装选项

  • 登陆地址和管理界面链接

  • 通过仪表盘发表、管理文章

  • 如果访问域名出现的是Godaddy的停靠页面，首先空浏览器缓存历史，然后通过CMD命令提示符输入ipconfig /flushdns刷新本地DNS

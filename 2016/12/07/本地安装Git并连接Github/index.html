本地安装Git并连接Github

检查本地是否安装

  • Mac和Linux系统使用Terminal工具、Window系统使用CMD输入git –version，如果成功安装显示如下并直接进入步骤三，否则进入步骤二。

安装

  • Windows平台

点击Git下载链接下载所需要的版本并按照如下步骤开始安装。

  • Linux平台

以Centos6.5为例，通常系统会默认自带git,版本为1.7.1，不够友好，所以首要任务就是卸载git。

  • Mac平台

MAC一般自带Git2.5.4版本不需安装。

连接Github

  • Windows平台

a.设置 user.name 和 user.email

b.生成ssh密匙文件（中间过程点击回车键）
最终在C:\Users/user/.ssh文件夹中生成id_rsa（私匙）和id_rsa.pub（公匙）

参考如下结果：

c.将公匙添加至GitHub

使用记事本打开id_rsa.pub，复制里面的公匙粘贴至https://github.com/settings/keys，然后选择New SSH key, title可不填写，将复制的公匙粘贴到key中再点击Add SSH key。


d.检测连接是否成功

在CMD中输入ssh git@github.com，成功结果如下：

  • Mac平台

a.在Terminal中运行ls ~/.ssh检查是否存在ssh密匙文件id_rsa（私匙）和id_rsa.pub（公匙）。

如果没有则在Terminal中运行ssh-keygen -t rsa -C “引号中填写注册Github邮箱”。

b.将公匙添加至GitHub

使用cat命令查看公钥内容，复制里面的公匙粘贴至https://github.com/settings/keys，然后选择New SSH key, title可不填写，将复制的公匙粘贴到key中再点击Add SSH key。



c.检测连接是否成功

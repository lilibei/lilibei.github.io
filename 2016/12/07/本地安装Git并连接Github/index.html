Windows系统

  • 设置user.name和user.email
  • 生成ssh密匙文件（中间过程点击回车键）
    最终在C:\Users/user/.ssh文件夹中生成id_rsa（私匙）和id_rsa.pub（公匙）

参考如下结果：

  • 将公匙添加至gitHub

使用记事本打开id_rsa.pub，复制里面的公匙粘贴至https://github.com/settings/keys，然后选择New SSH key, title可不填写，将复制的公匙粘贴到key中再点击Add SSH key。


  • 检测连接是否成功

macOS系统

  • 在Terminal中运行ls ~/.ssh检查是否存在ssh密匙文件id_rsa（私匙）和id_rsa.pub（公匙）。

如果没有则在Terminal中运行ssh-keygen -t rsa -C “引号中填写注册Github邮箱”。

  • 将公匙添加至GitHub

使用cat命令查看公钥内容，复制里面的公匙粘贴至https://github.com/settings/keys，然后选择New SSH key, title可不填写，将复制的公匙粘贴到key中再点击Add SSH key。



  • 检测连接是否成功

在Terminal命令行输入ssh -T git@github.com成功结果显示如下：

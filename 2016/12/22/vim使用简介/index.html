一般模式

光标移动

j: 向上移动
k: 向下移动
h: 向左移动
l: 向右移动
n+方向键: 向该方向移动n行
control+f: 向下翻一页
control+b: 向上翻一页
g: 进入首行
gg: 进入末行
shift+$: 进入行末
shift+^: 进入行末

复制剪切粘贴

x: 删除
n+x: 删除n个字符
dd: 删除整行
p或者shift+p: 粘贴
u: 撤销上一步命令
control+r: 反撤销
yy: 复制
n+dd或者n+yy: 删除n行或者复制n行
v: 进入可视化操作

编辑模式

i或者a或者o: 进入编辑模式
esc: 退出编辑模式

命令行模式

wq: 保存退出
w: 保存
q: 退出
q!: 不保存退出
set nu 显示行号
set nonu 不显示行号
nohl: 不高亮显示
/A08: 搜索带有A08的字符
1,20s/A08/aaa/g: 将1-20行中的A08替换为aaa #g代表全局替换
%s/A08/aaa/g 从首行到尾行进行替换
